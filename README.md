# Static-Type-Checker
Static Type Checker Java

## What you need to do
1. You just need to check primitives for type compatibility
2. DO NOT FREAKING DO LOCAL VARIABLES BECAUSE THERE IS NO SUPPORT FOR THEM!! 
2. Check widening
 ### Numeric Primitives 
 - short
 - int
 - long
 - float 
 - double
 
 ### Textual Primitives 
 - byte 
 - char
 
 ### Boolean and null primitives
 - boolean 
 - null
